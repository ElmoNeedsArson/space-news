# Space Launches

Track upcoming rocket launches and space events from Obsidian.

## Features

- Sidebar panel for upcoming launches and events
- Launch countdowns, providers, locations, and status
- Filters for launch providers, locations, and event types
- Optional notice 15 minutes before tracked launches

## Use

Open Space Launches from the rocket icon or the command palette.

Settings are available under Community plugins > Space Launches.

## Attribution
